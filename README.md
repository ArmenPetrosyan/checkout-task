
### Checkout - test work

---
*Bootstrapped with React-create-app.*
#### Tools & Libs
- React
- redux
- redux-form
- Material UI @next
- SASS
- CSS in JS

#### How to run locally?
- yarn install
- yarn start

 Project will be running on **http://localhost:3000**

#### How to build project?
- yarn run build

#### How to run tests?
- yarn run test

#### How to watch style changes?
- yarn run watch-css

#### How to build styles?
- yarn run build-css
